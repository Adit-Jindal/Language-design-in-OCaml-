# Language-design-in-OCaml-

This contains the following operations:
Integers and Floats:
- a+b - Addition
- a-b - Subtraction
- a*b - Multiplication
- a/b - Division

Vectors:
- v+u - vector addition
- v-u - vector subtraction
- k*v - scalar multiplication (scalar comes first)
- v.u - dot product
- |v| - magnitude
- dim v - dimensions
- v^u - angle between vectors
